## Project Overview
This project was completed as part of the Digital Literacy course. It focuses on understanding and applying essential digital skills required for academic and professional environments. The project covers topics such as digital awareness, professional online presence, coding platforms, communication etiquette, and cybercrime prevention.  

Through this project, I explored various digital tools and platforms, developed a basic digital portfolio, and gained practical knowledge about responsible and safe online behavior.  

---
